## 📊 HTML Tables

Tables are used to display data in rows and columns.

Main table tags:

- <table> → Creates the table
- <tr> → Table row
- <th> → Table heading (bold & centered)
- <td> → Table data cell

Tables are useful for:
- Timetables
- Marksheets
- Product information
- Data comparison

## 📝 HTML Forms

Forms are used to collect user input.

Examples:
- Login forms
- Registration forms
- Contact forms
- Search bars

Main form tag:
- <form> → Container for all input fields

---

### Common Input Types

- text → Normal text input
- password → Hidden characters
- email → Email input
- number → Numeric input
- date → D\er
- submit → Submit button

---

### Labels

The <label> tag describes an input field.
It helps users understand what to enter.

Forms allow interaction between user and website.
