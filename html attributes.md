## 🧩 What are HTML Attributes?

Attributes provide extra information about HTML elements.

They are always written inside the opening tag.

---

## 🛠 Basic Syntax

<tag attribute="value"> Content </tag>

Example:

<a href="https://google.com">Visit Google</a>

Here:
- href = attribute
- https://google.com = value

---

## ⭐ Common HTML Attributes

### 1️⃣ href
Used in anchor (<a>) tag to define link destination.

<a href="https://example.com">Click Here</a>

---

### 2️⃣ src
Used to define the source of an image, video, etc.

<img src="image.jpg">

---

### 3️⃣ alt
Alternative text for images (important for accessibility).

<img src="image.jpg" alt="Nature Image">

---

### 4️⃣ class
Used for styling and  elements in CSS.

<p class="text-style">Hello World</p>

---

### 5️⃣ id
Unique identifier for ant.

<div id="header"></div>

---

### 6️⃣ target
Defines where to open the linked document.

<a href="https://google.com" target="_blank">Open in new tab</a>

---

## 🎯 Why Attributes are Important?

- Control behavior of elements
- Connect HTML with CSS
- Connect HTML with JavaScript
- Improve SEO & accessibility

---

## 🧠 Pro Tip

Always use quotes around attribute values.
Example: class="box" ✅
