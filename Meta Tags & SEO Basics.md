## 🌐 What are Meta Tags?

Meta tags provide information about the webpage.

They are written inside the <head> section.

They are not visible on the page but important for browsers and search engines.

---

## ⭐ Important Meta Tags

### 1️⃣ Charset

Defines character encoding.

<meta charset="UTF-8">

---

### 2️⃣ Viewport (Responsive Design)

Makes website mobile-friendly.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

---

### 3️⃣ Description

Short description of the webpage (important for SEO).

<meta name="description" content="Learn HTML basics step by step.">

---

### 4️⃣ Keywords

Defines keywords related to page content.

<meta name="keywords" content="HTML, web development, beginner tutorial">

---

### 5️⃣ Author

Defines the author of the page.

<meta name="author" content="Your Name">

---

## 🎯 Why Meta Tags Matter?

- Improve search engine ranking
- Make website mobile responsive
- Help Google understand your 
- Improve click-through rate in search results

---

## 🧠 Example Head Section

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta name="description" content="HTML beginner tutorial">
  <meta name="keywords" content="HTML, beginner, web development">
  <meta name="author" content="Your Name">
  <title>HTML Basics</title>
</head>
