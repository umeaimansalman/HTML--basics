# HTML Basics – Beginner Notes

This repository contains beginner-friendly notes on HTML.  
These notes are part of my learning journey in web development.

---

## 📘 What is HTML?

HTML stands for **HyperText Markup Language**.  
It is used to create the **structure of web pages**.

HTML tells the browser:
- where headings should appear
- where paragraphs should be placed
- where images and links should be displayed
- how content is organized on a page

HTML is **not a programming language**.  
It is a **markup language** used to structure content.

---

## 🎯 Why Do We Use HTML?

Without HTML:
- A website cannot have structure.
- The browser cannot understand how to display content.

HTML is:
- Easy to learn
- Beginner-friendly
- The foundation of every website

---

## 🌍 Where is HTML Used?

HTML is used in:
- Websites
- Web aptions
- Email templates
- Online forms

---

## 🧱 Basic Structure of an HTML Document

An HTML document is divided into three main parts:

- `<html>` → Root element of the page
- `<head>` → Contains meta information (title, links, etc.)
- `<body>` → Contains visible content of the webpage

---
## 🧠 Important Learning Note

> HTML provides structure, CSS adds design, and JavaScript adds functionality.
